Problem-Solving Reflections


Blocked on a Simple Problem

  • Problem: Struggled with making a case-insensitive search in an array of strings.
  • Techniques: Re-read code, simplified the problem, used console logging, and googled.
  • Feelings: Frustration turned to focus and relief.
  • Lesson: Even simple problems need careful attention and all available tools.

    • Elegantly Solved a Problem

  • Problem: Needed to find an object in an array and return a specific property.
  • Techniques: Used `.find()` for a clean, reusable solution.
  • Feelings: Confident and satisfied.
  • Lesson: Simplicity and efficiency often go hand in hand.

    • Confidence in Problem-Solving Techniques

  • Pseudocode: Moderately confident—useful for structuring thoughts.
  • Trying something: Very confident—experimenting is key.
  • Rubber ducky method: Fairly confident—talking through problems helps.
  • Reading error messages: Highly confident—first step in troubleshooting.
  • Console.logging: Extremely confident—vital for understanding code.
  • Googling: Very confident—reliable for research.
  • Asking peers: Moderately confident—value input but sometimes hesitate.
  • Asking coaches: Moderately confident—seek help when truly stuck.
  • Reflecting: Somewhat confident—could reflect more consistently.

    • Reluctant to Ask for Help

  • Situation: Hesitated to ask for help on tricky logic, fearing it would make me seem less competent.
  • Next time: Remember that asking for help is normal and reach out sooner.
